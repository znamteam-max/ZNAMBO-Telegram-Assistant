## What Changed

- Added shared natural-language session routing before edit/setup handlers.
- Cancel phrases clear active item edit, reminder setup, recurring draft, and external calendar
  edit sessions.
- New global weekly/monthly reminder creation requests escape stale sessions and continue through
  mandatory OpenAI planning.
- Reminder setup now keeps collected fields across turns.
- End-of-day setup replies such as `do kontsa dnya` and `do 23.59` resolve to local `23:59`.
- Full reminder setup phrases can apply immediately without another end-time prompt.
- Added safe `/admin_repair_v2110 preview|apply`.
- Improved `/debuglast` with a safe suggested next prompt for recurring tool failures.
- No database migration was required.
