# ZNAMBO Telegram Assistant V2.4.0

Status: released

## Target behavior

- Keep one current Live Plan Dashboard at the bottom of the Telegram chat.
- Retire old dashboard, reminder and item-menu cards without blocking core mutations.
- Store reminder intent as policies and occurrences instead of unrelated reminder piles.
- Deliver an Event Reaction Menu after events instead of a text-only "как прошло?" prompt.
- Support before-event, post-event-menu, interval-window, recurring, nag-until-ack and long-term policies.
- Keep V2.3 mandatory OpenAI execution and typed item updates.
