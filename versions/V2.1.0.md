# ZNAMBO Telegram Assistant V2.1.0

Status: released

## Scope

- Jarvis management views and saved task-view state.
- Safe plan reset, cleanup and undo operations.
- Production reminder scheduler moved to cron-job.org.
- Protected production repair and reminder smoke checks.
- Targeted cleanup of legacy polluted planner records.

## Production proof

- Health endpoint returned the active deployment commit.
- Telegram webhook had no pending updates or last error.
- cron-job.org called `POST /api/reminders/run`.
- `/remindertest 2` produced a real Telegram reminder.
- Automatic reminder delivery was confirmed without manually invoking the runner.

## Known limitations

- Natural-language planner decisions could still bypass OpenAI or silently use a heuristic fallback.
