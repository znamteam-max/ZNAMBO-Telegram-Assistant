## Implemented

- Incomplete canonical recurring reminder policies are blocked before action plans, planner items,
  reminder policies, or reminders can be saved.
- Recurring draft sessions now store deterministic fingerprints and dedupe repeated incomplete
  requests.
- Global creation-intent session escape preserves recurring drafts for dedupe; `/cancel` still
  clears active sessions.
- Legacy V2 execution path now uses the same missing-time recurring draft guard as Jarvis.
- Item edit context supports all-day reschedules such as `сегодня целый день`.
- Reminder snooze writes `assistant.reminder_snooze_attempt` audit rows and falls back to item
  snooze when policy snooze cannot resolve.
- Undo normalizes active scheduled event-like task snapshots back to `event`.
- Added `/actionlog`, `/actionlog 24h`, `/actionlog 50`, `/actionlog export`, `/debugrecent`.
- Added `/admin_repair_v2130 preview|apply` plus protected admin repair API actions.
- No database migration was required.
