# V2.6.0 - Plan UI and Yandex Inbound Calendar

Release date: 2026-06-12

## Implemented

- Plan rows no longer show red/green/yellow urgency circles; visible star/fire importance is user-controlled.
- Item cards use a compact default menu, with technical calendar/debug actions under `Ещё`.
- Entity cards prefer in-place Telegram edits when the callback message is editable.
- Persistent Telegram bottom navigation provides Plan, Add, Tasks, Reminders and Settings.
- Compound item edits preserve explicit start/end time ranges, support same-day evening disambiguation,
  rename without quotes, and show full old/new previews.
- Added inbound Yandex CalDAV `REPORT calendar-query`, ICS parsing, weekly recurrence expansion,
  deduplication against JARVIS-synced objects, external-event cache, Plan merge, cards, hide/delete,
  and same-object-URL editing for non-recurring external events,
  `/calendar_sync`, `/calendar_import_status`, and bounded background import.
- Yandex failures remain best-effort and never block the reminder runner.
