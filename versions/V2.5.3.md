# V2.5.3 - Production Repair Enforcement and CalDAV Verification

Date: 2026-06-12

V2.5.3 is a focused production hotfix on top of V2.5.2. It does not replace the OpenAI
ActionPlan, scheduler, reconciler, runner lock, or anchor-grid reminder engine.

## Implemented

- Added canonical `buildUserTimelineView` for tasks, dashboard, reminders, and long-term views.
- Added the `unresolved_past` bucket for active work older than 48 hours.
- Strengthened `/admin_repair_v252 preview|apply` and made the repair idempotent.
- Added safe `/cleanup_garbage` preview and `/admin_state_v252`.
- Yandex CalDAV sync now verifies writes with a read-back GET.
- Added safe calendar error classes, `/calendardebug`, and `/calendar_test`.
- Event cards and planner mutation replies now show calendar sync status.
