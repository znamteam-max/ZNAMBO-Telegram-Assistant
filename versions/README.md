# ZNAMBO Telegram Assistant versions

Each production version has a separate immutable summary file in this directory.

Rules:

- one file per released semantic version;
- include scope, behavior changes, production proof, tests and remaining limitations;
- never include secrets, tokens, passwords, API keys, bearer tokens or database URLs;
- detailed chronological notes remain in `PROJECT_HISTORY.md` and `CHAT_HISTORY.md`.

Version index:

- `V2.0.0.md` - Smart AI Planner foundation.
- `V2.1.0.md` - production rollout and reliable reminder delivery.
- `V2.2.0.md` - mandatory observable OpenAI agent execution.
- `V2.3.0.md` - contextual completion, hybrid planning and typed item updates.
- `V2.4.0.md` - Live Plan Dashboard, Reminder Policy Engine and clean Telegram message lifecycle.
- `V2.4.1.md` - policy reconciliation, interval reliability, scheduler health and production legacy repair.
