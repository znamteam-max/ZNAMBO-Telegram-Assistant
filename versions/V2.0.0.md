# ZNAMBO Telegram Assistant V2.0.0

Status: released

## Scope

- Postgres conversation history and transcripts.
- Memory facts, summaries and retrieval context.
- Multi-action `ActionPlan` and `action_plan_items`.
- Events, tasks, preparation, tentative events, training and recurring reminders.
- Repeat-until-ack reminder model.
- Best-effort calendar synchronization.

## Verification

- V2 database migration applied to production Neon.
- Telegram webhook and health endpoint deployed on Vercel.
- Planner and reminder behavior covered by automated tests.

## Known limitations

- Natural-language routing still contained deterministic and heuristic paths.
- OpenAI execution was not yet proven for every natural-language turn.
